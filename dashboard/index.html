SSE — Claude CodeStreamable HTTP — Claude.ai web
📋 How to add to Claude.ai (web)
1
Make sure Meta Ads is connected on the Connections tab.
2
Open Claude.ai → Settings → Integrations and click Add MCP Server.
3
Paste your personal MCP URL above, name it SyncMetrics, click Save.
4
Start a new chat and ask: What are my Meta ad accounts?
🖥️ How to add to Claude Code (CLI)
1
Run: claude mcp add syncmetrics --transport sse <your-url>
2
Or edit ~/.claude/settings.json and add your URL under mcpServers.

Looker Studio

Pull your Meta Ads data into Looker Studio dashboards and reports

📊 How to connect Looker Studio
1
Make sure Meta Ads is connected on the Connections tab first.
2
Click the button below — the SyncMetrics connector opens directly in Looker Studio.
3
Click Sign in with SyncMetrics and log in with Google.
4
Your ad accounts load automatically. Select an account and click Connect.
5
Click Create Report — your Meta Ads data is ready to visualize!
